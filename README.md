## Sobre mim

Atuo atualmente como **Estagiário de Infraestrutura de TI na Fast Sistemas Construtivos**, prestando suporte aos usuários e auxiliando na manutenção do ambiente tecnológico da empresa.

Minha experiência envolve diagnóstico e resolução de problemas relacionados a hardware, software, sistemas Windows, conectividade, redes locais, impressoras, periféricos, acesso remoto e gestão de chamados.

Também desenvolvo projetos e automações utilizando **Python**, combinando conhecimentos em infraestrutura, suporte técnico, análise de dados e desenvolvimento backend.

Curso **Análise e Desenvolvimento de Sistemas na UNISUAM**.

* Suporte técnico, Service Desk e troubleshooting
* Sistemas Windows e Linux
* Redes locais, hardware, impressoras e acesso remoto
* Gestão e acompanhamento de chamados
* Automação de rotinas com Python
* Análise de dados com SQL, Excel, Pandas e NumPy
* Desenvolvimento backend com Python e Django
* APIs, bancos de dados e web scraping

## Projetos em destaque

### Taskfy — Gerenciador de Tarefas

Aplicação desktop desenvolvida com **Python, PySide6, SQLite e Matplotlib**.

Principais funcionalidades:

* Cadastro, edição e exclusão de tarefas
* Organização de tarefas por categorias
* Relacionamento entre tarefas e categorias
* Persistência local utilizando SQLite
* Pesquisa e filtros
* Visualização de tarefas concluídas e pendentes
* Gráfico de progresso
* Interface desenvolvida com PySide6
